Addendum

Process: I planned a six-slot story about a wild adventure, choosing word types (proper noun, noun, verb, adjective, place, number) that create humorous combinations.

Form design: The form uses method="POST" so inputs aren't exposed in the URL. Each input is paired with a label using matching for/id attributes for accessibility.

Server-side: The POST handler destructures req.body, validates all fields, then uses template literals to build the story and send it back as HTML.

Styling: Bold neo-brutalist aesthetic with a warm orange palette and thick drop-shadows to match the fun nature of a mad lib.
